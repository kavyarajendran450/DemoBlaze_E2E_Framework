DemoBlaze Playwright Automation Framework
This project demonstrates an automation framework built using Playwright with TypeScript.

Framework Design:

Page Object Model (POM)
Wrapper methods for reusable actions
Test data management using JSON
End-to-End automation scenario
Automated Scenario:

Login to Demoblaze
Navigate to Laptops category
Move to next page
Select MacBook Pro
Add product to cart
Verify product in cart
Delete product
Logout
Tech Stack:

Playwright
TypeScript
Node.js
Page Object Model
Run Tests: npx playwright test
